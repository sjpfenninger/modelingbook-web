Modeling the environment

Interactive Models DRAFT - do not publicize

Existing examples so far:

  • Lotka-Volterra model
  • Two-box model

Created by Stefan Pfenninger. The following libraries and resources are used: JSXGraph for calculations and graphing, Tangle for interactive adjustment of the parameters, and MathJax for math rendering. Box connections are rendered with jsPlumb. The CSS is based on Twitter Bootstrap. Some artwork from the Noun Project. The code for graphing the Lotka-Volterra equations is based on the JSXGraph Lotka-Volterra example.
